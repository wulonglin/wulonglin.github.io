应用介绍

App Introduction

上岛记 是一款专为 iOS 和 watchOS 平台设计的实用工具应用，让取餐码、排队号、快递柜码等重要信息快速显示在 iPhone 灵动岛和 Apple Watch 上，方便用户随时查看，解放双手。

Pin2Island is a practical utility app designed for iOS and watchOS platforms, allowing pickup codes, queue numbers, locker codes and other important information to be quickly displayed on iPhone Dynamic Island and Apple Watch, making it easy for users to check anytime with hands free.

核心特点

Core Features

  • 多场景支持：覆盖取餐码、排队号、快递柜码等多种生活场景需求
  • Multi-scenario Support: Covers various life scenarios including pickup codes, queue numbers, and locker codes
  • 纯本地运行：所有功能在设备本地执行，无需网络连接，保护用户隐私
  • Local Operation Only: All functions run locally on the device, no network connection required, protecting user privacy
  • Apple Watch 深度集成：通过 WCSession 实现 iPhone 与 Apple Watch 的无缝数据同步
  • Deep Apple Watch Integration: Seamless data synchronization between iPhone and Apple Watch via WCSession
  • 智能识别技术：采用 CoreML 机器学习模型，实现高效准确的文本和图像识别
  • Intelligent Recognition: Utilizes CoreML machine learning models for efficient and accurate text and image recognition
  • 低资源占用：优化的算法设计，确保应用运行流畅且省电
  • Low Resource Usage: Optimized algorithm design ensures smooth operation and battery efficiency
  • 简洁直观界面：专注于核心功能，提供无干扰的用户体验
  • Clean and Intuitive Interface: Focused on core functions, providing a distraction-free user experience
  • 实时数据同步：设备间数据实时更新，确保信息一致性
  • Real-time Data Sync: Real-time data updates between devices, ensuring information consistency

常见问题

Frequently Asked Questions

上岛记 是否需要网络连接？

Does Pin2Island require internet connection?

不需要。上岛记 是一款纯本地应用，所有功能均在设备本地运行，不依赖网络连接。

No. Pin2Island is a local-only app. All functions run locally on the device without requiring network connection.

应用如何在 iPhone 和 Apple Watch 之间同步数据？

How does the app sync data between iPhone and Apple Watch?

我们使用 Apple 提供的 App Groups 和 WCSession 技术进行设备间的本地数据同步，确保数据安全且无需云服务。

We use Apple's App Groups and WCSession technologies for local data synchronization between devices, ensuring data security without requiring cloud services.

应用会收集我的个人信息吗？

Does the app collect my personal information?

不会。上岛记 严格保护用户隐私，不收集、不上传任何用户数据或个人信息。

No. Pin2Island strictly protects user privacy and does not collect or upload any user data or personal information.

如何将应用更新到最新版本？

How to update the app to the latest version?

您可以通过 App Store 的自动更新功能或手动检查更新，获取 上岛记 的最新版本。

You can get the latest version of Pin2Island through App Store's automatic update feature or by manually checking for updates.
