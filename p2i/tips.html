自动提取取件码教程

Auto Extract Pickup Code

  1. 在「设置」页点击「添加快捷指令」安装「自动上岛记」
  2. 选择触发方式：操作按钮 / 轻点背面 / 辅助触控
  3. 在快递短信或取件页面触发，自动提取取件码（支持三段式、两段式、5-8位纯数字、字母前缀）
  4. 识别成功自动写入「历史」，可一键发送到实时活动与灵动岛
  5. 遇到长运单号、隐私号或被遮掩场景，系统会自动过滤干扰以提升准确性
  1. In “Settings”, tap “Add Shortcut” to install “Auto PinToIsland”
  2. Choose a trigger: Action Button / Back Tap / AssistiveTouch
  3. On courier SMS or pickup pages, trigger to auto-extract the code (supports tri-segment, two-segment, 5–8 digit numeric, alphanumeric prefix)
  4. Successful recognition is saved to History; send to Live Activities/Dynamic Island with one tap
  5. Long waybill numbers, privacy numbers, or masked cases are auto-filtered to improve accuracy

快速上手

Quick Start

添加「自动上岛记」快捷指令

Add the “Auto PinToIsland” Shortcut

  1. 在应用的「设置」中点击「添加快捷指令」跳转到快捷指令 App
  2. 完成添加后，设置触发方式：
  3. iPhone 15 Pro 及以上：系统设置 → 操作按钮 → 快捷指令 → 选取「自动上岛记」
  4. 其他机型：系统设置 → 辅助功能 → 触控 → 轻点背面 → 轻点两下选择「自动上岛记」
  5. 或小白点：系统设置 → 辅助功能 → 触控 → 辅助触控 → 自定操作 → 轻点一下选择「自动上岛记」
  1. In Settings, tap “Add Shortcut” to open Shortcuts app
  2. After adding, set a trigger method:
  3. iPhone 15 Pro and later: Settings → Action Button → Shortcuts → select “Auto PinToIsland”
  4. Other models: Settings → Accessibility → Touch → Back Tap → Double Tap → choose “Auto PinToIsland”
  5. Or AssistiveTouch: Settings → Accessibility → Touch → AssistiveTouch → Custom Actions → Single-Tap → choose “Auto PinToIsland”
提示：识别由系统快捷指令本地完成，App 仅接收结果，无需网络，隐私安全。
Note: Recognition runs locally via Shortcuts; the app only receives results, no network required, privacy-safe.

开始使用

Using the App

触发识别并保存到历史

Trigger Recognition and Save to History

  1. 在餐饮、排队、快递或票券页面，按压操作按钮/轻点背面/点小白点触发
  2. 成功识别后自动写入「历史」，可推送到实时活动与灵动岛
  3. 票券支持识别日期、影厅及座位；取餐/排队/快递支持识别号码
  4. 点击历史项可查看截图或二维码，支持「查看原图」
  1. On food/queue/courier/ticket screens, trigger via Action Button/Back Tap/AssistiveTouch
  2. Recognized results are saved into History and can be sent to Live Activities/Dynamic Island
  3. Tickets support date, hall and seats; food/queue/courier support numbers
  4. Tap a history item to view the screenshot or QR, with “View Original”

历史页技巧

History Tips

筛选与分组

Filtering and Grouping

  • 顶部可按「全部/餐饮/快递/票券/排队」筛选记录
  • Filter by “All/Food/Courier/Tickets/Queue” at the top
  • 自动按「今天/昨天/日期」分组，便于查找
  • Auto group by Today/Yesterday/Date for easier browsing

手势操作

Gestures

  • 右滑「发送到活动」，同步到灵动岛与锁屏
  • Swipe right to “Send to Activity” for Dynamic Island/Lock Screen
  • 左滑「删除」，从历史移除并可同时关闭对应活动
  • Swipe left to delete; related Live Activity can also end
  • 点击记录可打开预览卡片，大图或二维码独立查看
  • Tap to open preview sheet for full image or QR view

票券展示

Tickets Display

  • 标题自动合并座位：如「5排5·6座」
  • Title merges seats, e.g., “Row 5 Seats 5·6”
  • 显示日期与影厅；自提/券码场景显示有效期
  • Shows date and hall; voucher/self-pick shows expiry

笔记与设置

Notes and Settings

上岛笔记

Island Notes

  • 在「笔记」页输入 20 字内内容，支持复制、编辑、删除
  • Enter up to 20 chars in “Notes”; supports copy, edit, delete
  • 可发送到实时活动，同步到 Apple Watch
  • Send to Live Activities and sync to Apple Watch

实时活动与通知

Live Activities and Notifications

  • 可启用「自动关闭取餐活动」、显示/隐藏锁屏关闭按钮
  • Enable auto-close for meal activities, show/hide lock screen buttons
  • 「识别成功推送通知」可开关，默认开启
  • Toggle “Recognition Success” notifications; enabled by default

二维码设置

QR Settings

  • 支持「自动裁切二维码」，不便识别时可点击查看原图
  • Enable “Auto Crop QR”; use “View Original” when recognition is hard
  • 可一键清空历史记录（不可恢复）
  • Clear all history with one tap (irreversible)

Apple Watch

Apple Watch

  • 查看票券详情（场次、影厅、座位），实时活动自动同步
  • View ticket details (date, hall, seats); Live Activities auto-sync
  • 支持二维码在手表显示；抬腕即可取票/取餐
  • QR shows on watch; raise wrist to access tickets/pickups

识别优化

Recognition Optimization

  • 尽量截屏或使用官方页面展示的号码，避免被遮挡
  • Prefer screenshots or official pages with clear numbers
  • 票券支持识别「有效期至」「影厅」「座位」等关键词
  • Tickets detect “Expiry”, “Hall”, and seat patterns
  • 若灵动岛/状态栏数字干扰，重试或切换页面再触发
  • If status bar numbers interfere, retry or switch screen then trigger

隐私与安全

Privacy and Safety

  • 识别与截屏均由系统快捷指令本地完成，App 无网络依赖
  • Recognition and screenshots via Shortcuts locally; no network required
  • 应用不采集个人信息；历史可随时删除或清空
  • No personal data collection; delete or clear history anytime

常见问题解决

Troubleshooting

识别不准确

Inaccurate Recognition

  • 尝试更换页面或滚动，让号码更接近屏幕中央
  • Switch screens or scroll so the code sits near the center
  • 票券请确保包含日期/影厅/座位等字段
  • Tickets should include date/hall/seat fields clearly
  • 必要时点击「查看原图」关闭裁切后再试
  • If needed, use “View Original” and retry without cropping

Apple Watch 同步问题

Apple Watch Sync Issues

  • 确保手表与手机距离不远
  • Ensure watch and phone are within close range
  • 检查手表蓝牙连接
  • Check Bluetooth connection on watch
  • 在手表上重新启动应用
  • Restart the app on the watch

灵动岛不显示

Dynamic Island Not Showing

  • 确认 iPhone 型号支持灵动岛
  • Confirm iPhone model supports Dynamic Island
  • 检查灵动岛是否被其他应用占用
  • Check if Dynamic Island is occupied by other apps
  • 重启设备后重试
  • Restart device and try again
  • 检查是否启用实时活动：设置页「实时活动」开关需打开
  • Ensure Live Activities are enabled in Settings

快捷指令未触发

Shortcut Not Triggering

  • 在「设置」中重新点击「添加快捷指令」并确认触发方式
  • Tap “Add Shortcut” in Settings again and reconfigure triggers
  • iPhone 15 Pro 检查操作按钮绑定，其他机型检查轻点背面/小白点设置
  • Check Action Button (15 Pro) or Back Tap/AssistiveTouch (others)
